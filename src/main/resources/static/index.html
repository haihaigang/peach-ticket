1.天津地铁乘车导航原则

乘客进站时，禁止携带违禁物品，合理使用地铁设施。

乘客购票时，了解地铁相关票务政策，遵守自动售票机使用规定。

乘客入闸时，正确使用闸机，遵守相关票务规定。

乘客候车时，遵守相关安全管理规定。

乘客乘车时，注意上下车安全，文明乘坐。

乘客出闸时，正确使用闸机，遵守相关票务规定。

2.天津地铁乘车导航

2.1乘客进入地铁车站时

禁止携带易燃，易爆，有毒，有放射性等危险物品。

禁止携带腐败，变质，有异味的物品以及宠物。

携带大型行李的乘客请勿乘坐地铁。

禁止在地铁车站内随意张贴，摆摊设点，从事未经允许的商业，娱乐活动，以及乞讨行为。

禁止在地铁车站内奔跑，嬉戏。

禁止在地铁车站，车厢内吸烟。

禁止擅自进入地铁非公共区域。

身患传染性疾病和酒醉者请不要进站

乘客在使用自动扶梯时，应靠近扶梯右侧站立，握紧扶梯扶手，小心摔倒。

避免贴近自动扶梯边缘，以免脚或衣物被卡在夹缝中。

身患疾病，及老，幼，病，残，孕等人士，乘坐自动扶梯应有人陪同。

禁止非法操作自动扶梯的紧急停机按钮。

2.2乘客购票时

乘客到自动售票机处购买单程票，按照自动售票机使用规则正确操作，禁止损坏自动售票机。

乘客在自动售票机上购买车票时，如遇到异常现象不要擅自解决，可找工作人员帮助解决。

乘客可到车站客服中心处办理储值卡的购买和充值业务。

乘客可免费带领一名身高1.1米以下（含1.1米）儿童免费乘车，带领两名以上儿童者，其他儿童应另行购票。身高1.1米以下（含1.1米）的儿童不能单独乘车。

2.3乘客入闸时

乘客应持有效的车票入闸，禁止翻越，攀爬闸机。

乘客乘车应一人一票，禁止任何形式的逃票行为。

乘客从入闸开始，须在120分钟之内搭乘完地铁，超出120分钟尚未出闸的视为超时，超时须按最高单程票价补交超时车费。

乘客应按实际车程够买车票，如乘客所使用的车票，不足以支付所达到车站的实际车费时，须补交超程车费。

乘客在入闸后遗失车票须到车站客服中心处办理补票业务。

乘客在入闸以后，禁止饮食。
